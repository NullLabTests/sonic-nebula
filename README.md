# Sonic Nebula

A visually and audibly compelling web app that visualizes audio input in real-time using p5.js. Particles form a nebula that reacts to sound frequencies, creating an immersive experience.

## Features
- Real-time audio reactivity using microphone input
- 3D particle system in WebGL
- Dynamic color shifting based on audio
- Responsive design

## How to Run
1. Open `index.html` in a modern browser.
2. Allow microphone access.
3. Make some noise and watch the nebula respond!
